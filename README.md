## 1. 서비스 소개
### 해결하고 싶었던 문제
1. 모임 시간을 정하는데 `커뮤니케이션 비용`이 많이 들어감
2. 기존의 모임 시간을 정하는 서비스들은 `모바일에서 사용하기 불편함`
### 우리의 해결 방안
* **모바일에서 편하게 사용**할 수 있는 UXUI로 **모임 시간 투표 서비스**를 만들자!
